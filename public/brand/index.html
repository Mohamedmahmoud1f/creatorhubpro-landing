07 — Brand Rules

Do's & Don'ts

Critical rules to maintain brand consistency and premium perception across all touchpoints.

✅ Always Do This

  • Use the logo on dark backgrounds (preferred) — maintains premium feel
  • Use Gold (#F59E0B) only for CTAs, key stats, and the "Pro" wordmark — never as body text color
  • Maintain minimum logo clear space: equal to the height of the icon mark on all sides
  • Use Inter (900 / 800) for all English headlines — never use lighter weights for titles
  • Use Cairo for all Arabic text — ensures correct RTL rendering and visual consistency
  • Keep the gradient direction consistent: always 135° (top-left to bottom-right)
  • Use the dark color stack (#07050F → #1A1730) for all backgrounds — never pure black or white
  • Maintain 8px grid for all spacing decisions — 8, 16, 24, 32, 48, 64, 96px
  • Always pair primary violet buttons with gold CTA for conversion — dual-button pattern
  • Use emerald green (✓ checkmarks) exclusively for benefits, results, and trust signals

🚫 Never Do This

  • Never place the logo on a white background without adapting to the dark/colored version
  • Never rotate, skew, stretch, or distort the logo in any direction
  • Never use more than 3 brand colors in a single design element or card
  • Never use Gold as a background fill for large areas — it loses premium value
  • Never use system fonts (Arial, Helvetica) in branded content — Inter only
  • Never add drop shadows to the wordmark text directly — use glow only on the icon mark
  • Never use pure #000000 black or #FFFFFF white — always use the dark palette or near-white
  • Never apply the brand gradient to body text — only to display headlines and icon fills
  • Never use red (#EF4444) for anything except errors, urgency, and danger signals
  • Never mix the Arabic and English typography styles in the same line of body text
